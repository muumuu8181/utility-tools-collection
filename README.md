# Utility Tools Collection

実用的なデスクトップユーティリティツール集

## 📁 ツール一覧

### 🎨 desktop_apps/
デスクトップアプリケーション集
- **desktop_pet/** - デスクトップペット（透明ウィンドウ、ランダム移動）
- **image_animator/** - 画像アニメーションツール

### 🛠️ tools/
メディア処理ツール集
- **background_remover/** - 背景除去ツール（最適化版含む）
- **video_tools/** - MP4→GIF変換、透明GIF作成ツール

### 🔤 romaji_converter/
- 日本語→ローマ字変換Webアプリ
- kuroshiro.js使用

### 📋 FILE_ORGANIZATION_GUIDE.md
- ファイル整理のベストプラクティスガイド

## 🚀 使用方法

各ツールのディレクトリに移動して、Pythonまたはブラウザで実行してください。
